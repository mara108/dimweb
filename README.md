### Генератор русских диминутивов
## Основная идея проекта
Веб-приложение для автоматического поиска диминутивов русского языка на основе результатов
лингвистического исследования и языковой модели GPT.
Пользователь вводит существительное, после чего система формирует список возможных
диминутивных форм и отображает их характеристики.
